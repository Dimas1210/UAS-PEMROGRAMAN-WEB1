JAWABAN NO 1
Website adalah kumpulan halaman web yang saling terhubung dan dapat diakses melalui internet. Halaman web tersebut terdiri dari teks, gambar, video, dan berbagai elemen interaktif lainnya. Browser kemudian mengirimkan permintaan HTTP ke web server yang hosting website tersebut. Web server menerima permintaan HTTP dan mengirimkan respons HTTP ke browser. Browser menerima respons HTTP dan menampilkan halaman web kepada pengguna. Hubungan antara Web Server, Domain, dan Hosting Web server adalah komputer yang menjalankan perangkat lunak yang berfungsi untuk menyimpan dan mengirimkan halaman web. Domain adalah nama unik yang digunakan untuk mengidentifikasi sebuah website di internet. Hubungan antara web server, domain, dan hosting adalah sebagai berikut:
*Web server dan domain adalah dua hal yang berbeda, tetapi keduanya saling terkait. Domain adalah nama yang digunakan untuk mengidentifikasi web server. *Hosting adalah layanan yang menyediakan ruang penyimpanan dan bandwidth untuk website.
Web server dan hosting saling terkait karena web server membutuhkan ruang penyimpanan dan bandwidth untuk menyimpan dan mengirimkan halaman web. Web server Web server adalah komputer yang menjalankan perangkat lunak yang berfungsi untuk menyimpan dan mengirimkan halaman web. Web server software yang umum digunakan adalah Apache, Nginx, dan IIS. Web server menerima permintaan HTTP dari browser dan mengirimkan respons HTTP ke browser. Domain Domain adalah nama unik yang digunakan untuk mengidentifikasi sebuah website di internet. Nama domain adalah nama yang digunakan untuk mewakili website tersebut. org Hosting Hosting adalah layanan yang menyediakan ruang penyimpanan dan bandwidth untuk website. Ruang penyimpanan digunakan untuk menyimpan halaman web, file media, dan database. Bandwidth digunakan untuk mengirimkan halaman web kepada pengguna.
